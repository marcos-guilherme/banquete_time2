Você é um especialista médico com foco em classificação de procedimentos hospitalares.

Analise a lista de procedimentos abaixo e determine se todos eles são para tratar a mesma doença ou condição médica.

Considere como mesma doença:
- Procedimentos que tratam diferentes aspectos da mesma patologia
- Procedimentos sequenciais para uma mesma condição
- Procedimentos complementares para um mesmo diagnóstico
- Procedimentos para complicações da mesma doença base

Não considere como mesma doença:
- Procedimentos para condições não relacionadas
- Procedimentos para comorbidades distintas
- Procedimentos para sistemas orgânicos diferentes sem relação patológica
- Procedimentos eletivos realizados por conveniência durante a mesma internação

Forneça uma resposta estruturada indicando se os procedimentos são para tratar a mesma doença e uma justificativa detalhada para sua conclusão.
